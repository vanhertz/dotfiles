# Dotfiles

Configuración personal administrada con Chezmoi.

## Requisitos

- Git
- Chezmoi

## Instalación

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply TUUSUARIO
```

## Contenido

- Git
- Bash
- Fish
- Kitty
- Ghostty
- Fastfetch
- Btop
- Neovim
- KDE Plasma
- Starship
